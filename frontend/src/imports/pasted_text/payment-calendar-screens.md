ГРУППА 2 — Платёжный календарь (Экран 2)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Переключатель периода:
- Таб "Неделя"  → показывает 7 колонок дней
- Таб "Месяц"   → показывает 30-31 колонку (текущий вид)
- Таб "Период"  → появляется date range picker 
                  с полями "с" и "по"

Кнопки:
- "Создать заявку"       → открывает Экран 3 (модалка)
- "Сформировать реестр"  → переход на Экран 5 (реестр)

Фильтры (выпадающие списки):
- Счёт → показывает варианты: Все счета / 
  Расчётный счёт №1 / Расчётный счёт №2 / Касса
- Статья → показывает варианты статей
- Статус → все статусы заявок

Ячейка календаря:
- Клик на любую ячейку → открывает Экран 4 (drawer)
- Ячейка с кассовым разрывом при клике → 
  drawer открывается с красным предупреждением сверху
  "Кассовый разрыв: −260 000 ₽"

Стрелки периода:
- < стрелка влево  → переключает на предыдущий месяц
- > стрелка вправо → переключает на следующий месяц

ГРУППА 3 — Модалка "Создать заявку" (Экран 3)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Тогл "Повторять автоматически":
- Выкл (по умолчанию) → поля частоты скрыты
- Вкл → появляются поля "Частота" и "Дата окончания"

Радиокнопки приоритета:
- Клик на Высокий / Средний / Низкий → 
  выбранный вариант получает обводку

Кнопки:
- "Отправить на согласование" → 
  модалка закрывается, в правом нижнем углу 
  появляется toast "Заявка отправлена на согласование"
  фон #80A080, исчезает через 3 секунды
- "Сохранить черновик" → 
  модалка закрывается, toast 
  "Черновик сохранён" фон #A0A080
- "Отмена" / ✕ → модалка закрывается, 
  возврат на календарь

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ГРУППА 4 — Drawer карточки заявки (Экран 4)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Кнопки действий:
- "Согласовать" → 
  бейдж статуса меняется на "Согласована" (#80A080),
  кнопки "Согласовать" и "Отклонить" исчезают,
  в историю добавляется строка с решением
- "Отклонить" → 
  появляется поле "Введите причину отказа" + 
  кнопка подтверждения,
  после подтверждения бейдж → "Отклонена" (#C0504A)
- "Перенести дату" → 
  появляется date picker прямо в drawer,
  после выбора даты — toast 
  "Дата перенесена, остатки пересчитаны"
- ✕ закрыть → drawer закрывается, 
  возврат на календарь

Вкладки:
- "История согласований" → показывает таблицу истории
- "Журнал изменений"     → показывает лог правок

ГРУППА 5 — Реестр платежей (Экран 5)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Чекбоксы строк:
- Клик на чекбокс строки → 
  строка выделяется (фон #E0C0A0),
  счётчик "Выбрано: N заявок" обновляется,
  кнопка "Отметить оплаченными" становится активной

- Клик на чекбокс в шапке таблицы → 
  выделяются все строки

Кнопки:
- "Выгрузить в Excel" → 
  появляется toast "Файл формируется..." 
  затем "Реестр_18.06.2026.xlsx готов к скачиванию"
- "Отметить оплаченными" (активна при выборе) → 
  статус выбранных строк меняется на "Оплачена"
- "Добавить заявку в реестр" → 
  модалка с поиском по заявкам

ГРУППА 6 — Список заявок (Экран 7)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Фильтры:
- Поле поиска → при вводе текста таблица фильтруется
- Статус выпадающий → показывает только нужный статус
- Кнопка "Сбросить фильтры" → все фильтры очищаются

Действия в строке:
- ✏ Редактировать → открывается модалка Экрана 3 
  с заполненными полями (режим редактирования)
- → Отправить на согласование (для черновика) → 
  статус меняется на "На согласовании", 
  toast "Заявка отправлена на согласование"
- 🗑 Удалить (для черновика) → 
  появляется диалог подтверждения 
  "Удалить заявку № 2847?" 
  Кнопки: "Удалить" (#C0504A) и "Отмена"

Кнопки сверху:
- "Создать заявку"   → открывает модалку Экрана 3
- "Импорт из Excel"  → 
  появляется модалка с drag-and-drop зоной 
  для загрузки файла

Пагинация:
- Клик на номер страницы → активная страница 
  меняется (фон #80A080)
- ← / → → переключение страниц

ОБЩИЕ ПРАВИЛА ИНТЕРАКТИВНОСТИ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Toast-уведомления (появляются всегда в правом 
нижнем углу):
- Успех: фон #80A080, иконка ✓, текст белый
- Предупреждение: фон #E0C0A0, иконка ⚠, 
  текст #7A5A30
- Ошибка: фон #C0504A, иконка ✕, текст белый
- Все toast исчезают через 3 секунды
- Анимация: появление снизу вверх

Диалоги подтверждения:
- Фон #FAFAF5, рамка #C0C0A0, радиус 12px
- Текст вопроса Bold
- Кнопка подтверждения — цвет зависит от действия
- Кнопка "Отмена" всегда текстовая

Анимации переходов:
- Модалки: появление fade + scale от 95% до 100%
- Drawer: выезд справа (slide from right)
- Смена вкладок: fade между содержимым
- Toast: slide up + fade

Начни с Группы 1 (навигация) — 
она даст основу для всего остального.
После каждой группы пиши 
"Группа N завершена — готов к следующей".
